#Tic_Tac_Toe

2 players tic tac toe game on a 3x3 grid.

###Practice with:

* Object oriented programming
* Array, Hash table

###Sample output:

To run: download the rb file and run this ruby command on the console if you have ruby installed:

    $ ruby Tic_tac_toe.rb
